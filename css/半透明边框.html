# 想要实现的效果

背景是红色,内容背景是白色,内容的边框是透明的,透过边框能看到底部的红色

# 实现方式

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    body{
      background-color: red;
    }
    .box{
      width: 100px;
      height: 100px;
      margin:0 auto;
      background-color: #fff;
      border:10px solid hsla(0, 0%, 100%, 0.5);
      background-clip: content-box;// 实现的关键
    }
  </style>
</head>
<body>
  <div class="box"></div>
</body>
</html>
